# 沪深300PE平衡策略

## 策略概述

沪深300PE平衡策略是一种基于沪深300指数市盈率（PE）的股债动态平衡策略。该策略通过PE指标来判断市场估值水平，动态调整股票和债券的仓位配比。

## 策略原理

### PE与仓位关系

策略基于一个核心理念：**PE值越低，估值越低，应该加大股票仓位；PE值越高，估值越高，应该降低股票仓位。**

### 仓位规则

- **PE ≤ PE最小值（低估线）**：股票仓位最高（默认60%）
- **PE ≥ PE最大值（高估线）**：股票仓位最低（默认10%）
- **PE最小值 < PE < PE最大值**：线性插值后离散化为固定档位

### 仓位计算公式

```
连续值 = 最高仓位 - (PE - PE最小值) / (PE最大值 - PE最小值) × (最高仓位 - 最低仓位)
最终仓位 = 将连续值映射到最近的离散档位
```

### 调仓规则

- **检查频率**：每N个月（可配置，默认6个月）检查一次PE
- **调仓条件**：目标仓位与当前仓位差异超过1%时调仓
- **调仓方式**：立即调整至目标仓位

## 策略参数

所有参数均可在UI上配置：

### 1. PE最小值（低估线）
- **默认值**：11
- **说明**：PE小于等于此值时，采用最高股票仓位
- **调整建议**：
  - 适当降低：在更早的时候达到满仓，更激进
  - 适当提高：在更低估值时才满仓，更保守

### 2. PE最大值（高估线）
- **默认值**：16
- **说明**：PE大于等于此值时，采用最低股票仓位
- **调整建议**：
  - 适当降低：更早开始降低仓位，更保守
  - 适当提高：在更高估值时才降低仓位，更激进

### 3. 最低股票仓位
- **默认值**：10%
- **说明**：股票配置的最低比例
- **调整建议**：根据风险承受能力调整，偏保守可以降低

### 4. 最高股票仓位
- **默认值**：60%
- **说明**：股票配置的最高比例
- **调整建议**：根据风险承受能力调整，偏激进可以提高

### 5. 仓位档位数
- **默认值**：6档
- **说明**：从最低到最高仓位之间的离散档位数量
- **示例**：
  - 2档：仅10%和60%两个仓位
  - 6档：10%, 20%, 30%, 40%, 50%, 60%
- **调整建议**：更多档位提供更细粒度的仓位调整

### 6. 复查间隔
- **默认值**：6个月
- **说明**：每N个月检查一次仓位
- **调整建议**：
  - 缩短间隔：更频繁调仓，对市场变化反应更快
  - 延长间隔：降低交易频率，减少交易成本

## 策略优势

1. **量化指标**：基于客观的PE指标，避免主观判断
2. **动态平衡**：根据市场估值自动调整股债比例
3. **风险控制**：通过股债配置实现风险分散
4. **参数可调**：所有关键参数都可以根据个人风险偏好调整
5. **离散档位**：减少频繁调仓，降低交易成本

## 数据来源

- **股票数据**：沪深300ETF基金（510300）
- **估值数据**：沪深300基金的市盈率
- **债券数据**：10年期国债收益率

## 使用方法

访问页面：`/backtest/csi300-pe-balance`

### 操作步骤

1. 选择回测年限
2. 配置策略参数（或使用默认值）
3. 系统将自动：
   - 获取沪深300基金数据
   - 计算每日PE并执行回测
   - 展示策略表现和详细数据

## 策略表现指标

回测结果包含以下关键指标：
- 累计收益率
- 年化收益率
- 最大回撤
- 最终仓位
- 交易次数
- 年度详情

## 示例场景

### 保守型投资者
```
PE范围：10-18
仓位范围：5%-40%
档位数：5
复查间隔：3个月
```

### 均衡型投资者（默认）
```
PE范围：11-16
仓位范围：10%-60%
档位数：6
复查间隔：6个月
```

### 激进型投资者
```
PE范围：12-15
仓位范围：20%-80%
档位数：8
复查间隔：12个月
```

## 注意事项

1. 历史回测不代表未来表现
2. PE指标有一定滞后性
3. 调整参数时应考虑交易成本
4. 建议在充分理解策略逻辑后使用
5. 可以通过调整参数来适应不同市场环境
