# 个股估值分析（Single-Stock Valuation Analysis）

> 角色定位：本页是「价值投资系统」（见 `PRD_0.md`）中 **[选股池]** 环节的个股深度研究页，
> 输出标准化的「估值结论 + 击球区 + 看多/看空逻辑」，供选股池与仓位管理引用。
> 数据来源：定量指标由 `a-stock-data` Skill 提供（Skill 内部按优先级在 13 个数据源中自动选取，见 §11）；
> 定性分析（标记 `AI`）由大模型生成，并附引用来源与生成时间戳。

---

## 0. 目标与范围

- **一句话目标**：输入一个股票代码，输出一份"是否值得买、当前贵不贵、买点在哪、看多看空理由"的结构化研究报告。
- **核心问题**：买什么（质量）→ 何时买（估值分位 / 击球区）→ 为什么（多空逻辑）。
- **页面路由（建议）**：`/stocks/[code]`，例如 `/stocks/601088`（中国神华）。
- **数据标注约定**：
  - 🟦 `Data`：由 `a-stock-data` Skill 提供的客观数据，可回测、可量化。
  - 🟨 `AI`：模型生成的定性判断，需标注「生成时间 + 信息来源 + 置信度」，不可作为唯一决策依据。

---

## 1. 页面整体结构（Information Architecture）

```
┌─────────────────────────────────────────────────────────┐
│  顶部：股票搜索框 + 当前标的（名称/代码/行业/所属池）        │
│  概览卡片：现价 | PE-TTM | 历史分位 | 股息率 | 击球区状态     │
├─────────────────────────────────────────────────────────┤
│  左侧锚点导航（吸附）：                                      │
│   1 业务分析  2 核心运营  3 利润质量  4 财务指标            │
│   5 股东回报  6 政策监管  7 护城河  8 全球化  9 估值总结     │
├─────────────────────────────────────────────────────────┤
│  右侧主内容区：各分析模块（折叠卡片，可逐节展开）            │
├─────────────────────────────────────────────────────────┤

```

- 顶部概览卡：一屏内给出"贵不贵、买不买"的第一直觉。
- 每个模块统一为「**结论先行 → 图表 → 明细表 → AI 解读**」的卡片结构。
- 全页支持导出 PDF / 分享，作为研究存档。

## 2. 业务分析（Business）🟨 AI + 🟦 Data

**分析目标**：看懂公司靠什么赚钱、收入结构是否健康。

| 维度 | 内容 | 来源 |
|------|------|------|
| 业务模式 | Top3 业务线及其营收占比、毛利占比 | 🟨 AI（财报）+ 🟦 营收数据 |
| 收入分析 | 国内 / 国外收入拆分与同比趋势 | 🟨 AI（财报）|

**页面功能**：
- Top3 业务收入占比环形图 + 近 3~5 年趋势堆叠柱状图。
- 国内/海外收入双环对比，标注同比增速。

**UI 设计**：
- 双列卡片：左「业务结构」环形图，右「地区结构」环形图。
- 鼠标悬停显示金额、占比、同比；AI 段落给出一句话定性结论。

---

## 3. 核心运营指标（Operations）🟦 Data

**分析目标**：判断经营效率与投入产出。

- 关注：**成本 / 研发 / 销售** 三项费用率及其趋势。
- 衍生指标：毛利率、费用率（销售/管理/研发）、研发占营收比、人效。

**页面功能**：
- 三费率趋势折线图（同图叠加，可切换"绝对值/占营收比"）。
- 与行业均值对比的横向条形图（若有行业数据）。

**UI 设计**：
- 顶部 KPI 小卡（成本率 / 研发率 / 销售率，含同比箭头）。
- 下方多线趋势图，时间轴 1y/3y/5y/10y 可切换。

---

## 4. 利润质量 & 周期性（Profit Quality & Cyclicality）🟦 Data + 🟨 AI

**分析目标**：利润是否真实、是否处于周期高/低点。

| 维度 | 内容 | 来源 |
|------|------|------|
| 利润率 & 趋势 | 毛利率 / 净利率 / ROE 走势 | 🟦 Data |
| 现金含量 | 经营现金流 / 净利润 比值 | 🟦 Data |
| 周期性 | 是否周期股、当前处于周期什么位置 | 🟨 AI + 🟦 营收/利润波动 |

**页面功能**：
- 利润率多线趋势图（毛利率/净利率/ROE）。
- 周期定位：营收/利润同比增速波动图，AI 标注"当前位于周期 上行/顶部/下行/底部"。

**UI 设计**：
- "周期位置"用一个半圆仪表盘（底部→顶部）直观指示。
- 利润质量给一个评分徽章（A/B/C），点击展开计算依据。

---

## 5. 财务指标（Financials）🟦 Data

**分析目标**：基本面健康度体检。

- 核心指标：营收、净利润、毛利率、净利率、ROE、资产负债率、有息负债、经营现金流、自由现金流。
- 提供近 5~10 年的年度明细表 + 同比/复合增速。

**页面功能**：
- 财务总表（可排序、可切换年/季）。
- 关键指标趋势小图（sparkline）嵌入表格行。

**UI 设计**：
- 复用现有 `YearlyDetailsTable` 风格的明细表组件。
- 异常值（如负债率过高、现金流为负）红色高亮提示。

---

## 6. 股东回报（Shareholder Return）🟦 Data

**分析目标**：公司是否真金白银回报股东。

- 关注：**分红（股息率、分红率/派息比例）** 与 **回购**。
- 衍生：股息率历史分位、连续分红年数。

**页面功能**：
- 股息率历史走势图 + 当前分位标注。
- 分红率 / 回购金额逐年柱状图。

**UI 设计**：
- KPI 卡：当前股息率 + 历史分位 + 连续分红年数。
- "股息率溢价"提示：当前股息率相对自身历史与无风险利率（10年国债）的溢价，呼应击球区逻辑。

---

## 7. 政策、监管环境分析（Policy & Regulation）🟨 AI

**分析目标**：识别行业政策顺风/逆风与监管风险。

**页面功能**：
- AI 生成的政策环境摘要（利好 / 利空 / 中性分类）。
- 关键政策事件时间线。

**UI 设计**：
- 三色标签（顺风/中性/逆风）+ 要点列表。
- 每条结论附信息来源链接与生成时间戳；提供"重新生成"按钮。

---

## 8. 市场竞争力 / 护城河 / 技术创新（Moat & Innovation）🟨 AI

**分析目标**：评估长期竞争优势是否牢固。

- 框架：成本优势 / 网络效应 / 无形资产（品牌、专利）/ 转换成本 / 规模效应。
- 技术创新：研发投入产出、专利、核心技术壁垒（结合第 3 节研发数据）。

**页面功能**：
- 护城河雷达图（5 个维度评分）。
- AI 文字论证 + 主要竞争对手对比要点。

**UI 设计**：
- 雷达图 + 一句话"护城河强度"结论徽章。
- 可展开查看每个维度的判断依据。

---

## 9. 全球化、出海分析（Globalization）🟨 AI + 🟦 Data

**分析目标**：海外业务的成长性与风险。

- 结合第 2 节海外收入占比与增速。
- AI 评估：出海市场、地缘/汇率风险、海外竞争格局。

**页面功能**：
- 海外收入占比与增速趋势（复用第 2 节数据）。
- AI 风险/机会要点。

**UI 设计**：
- 简洁双栏：左数据趋势，右 AI 机会/风险清单。

## 10. 估值总结（Valuation Summary）🟦 Data + 🟨 AI ⭐核心输出

**分析目标**：把以上分析收敛为一句可执行的结论。

| 输出项 | 说明 | 来源 |
|--------|------|------|
| 当前估值 | PE-TTM / PB / 股息率 当前值 | 🟦 Data |
| 历史分位 | 当前 PE / 股息率 在历史区间的百分位 | 🟦 Data |
| 击球区状态 | 估值分位 + 股息率溢价是否进入"可击球"区间 | 🟦 规则 |
| 看多逻辑 | 3~5 条买入理由 | 🟨 AI（综合各模块）|
| 看空逻辑 | 3~5 条风险/卖出理由 | 🟨 AI（综合各模块）|

**击球区规则（建议初版，可配置）**：
- 强烈低估（击球区）：PE 历史分位 ≤ 20% 且 股息率分位 ≥ 80%。
- 合理：分位 20%~70%。
- 高估（观望）：PE 历史分位 ≥ 80%。

**页面功能**：
- PE-TTM 历史走势图 + 分位带（20/50/80 分位线）+ 当前点标注。
- 击球区状态大徽章（低估/合理/高估）+ 颜色信号灯。
- 看多 / 看空逻辑左右双栏卡片。
- 「加入选股池」「设置价格预警」操作按钮。

**UI 设计**：
- 顶部一个醒目的「击球区信号灯」（绿/黄/红）。
- 看多绿色卡 / 看空红色卡并排，每条带来源锚点（点击跳转对应模块）。

## 11. 数据来源与依赖

> **本页所有定量数据统一由 `a-stock-data` Skill 提供。** 具体用哪个底层数据源由 Skill 自行判断
> （A 股全栈数据工具包，7 层 / 27 端点 / 13 数据源，内部按封 IP 风险自动选源），本文档不指定具体接口。

### 主数据源：`a-stock-data` Skill 🟦
- 安装位置：`.cursor/skills/a-stock-data/SKILL.md`（项目级 Skill，随仓库共享）。
- 运行依赖：独立 Python venv，见 `.cursor/skills/a-stock-data/requirements.txt`（`mootdx / requests / pandas / stockstats`）。
- 选源策略（由 Skill 内部决定，详见 SKILL.md）：
  - **首选** mootdx（通达信，TCP）/ 腾讯财经（HTTP）—— 不封 IP：K 线、五档、实时价、PE/PB/市值/换手率、季报、F10、财报三表。
  - 其余（东财 / 同花顺 / 巨潮 / 新浪 / 百度 / iwencai）—— 用于龙虎榜、解禁、两融、大宗、股东户数、分红、资金流、研报、新闻、公告、北向资金等独有数据；东财系统一走内置限流 `em_get()` 防封。
- 各模块取数映射（端点选择由 Skill 判断）：
  | 本文档模块 | 所需数据 |
  |-----------|---------|
  | 2 业务分析 / 6 股东回报 | 季报字段、F10、分红送转历史 |
  | 3 核心运营 / 4 利润质量 / 5 财务指标 | 财报三表、季报快照、F10 |
  | 6 股东回报 / 10 估值（估值分位） | 实时 PE/PB/市值 + 历史 K 线/PE 序列 |
  | 8 护城河 / 9 全球化（佐证数据） | 研报一致预期、行业对比、概念板块归属 |
  | 资金/情绪（呼应 PRD_0 调节阀） | 资金流向、龙虎榜、融资融券、北向资金、解禁 |
  | 9/10 多空逻辑佐证 | 个股新闻、研报、公告 |
- 无风险利率：10 年期国债收益率（用于股息率溢价 / ERP，见 `docs/indicators.md`）。
- ⚠️ 限制：mootdx 走国内 TCP，海外服务器需代理；东财部分大陆住宅 IP 有间歇风控（重试/换网络）。

### AI 定性分析 🟨
- 大模型基于上述结构化数据 + 财报/公告/新闻生成，输出须带来源链接与生成时间戳。

### 接入方式（待定，第二步细化）
