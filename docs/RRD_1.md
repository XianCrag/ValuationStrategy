# 个股估值分析（Single-Stock Valuation Analysis）

> 角色定位：本页是「价值投资系统」（见 `PRD_0.md`）中 **[选股池]** 环节的个股深度研究页，
> 输出标准化的「估值结论 + 击球区 + 看多/看空逻辑」，供选股池与仓位管理引用。
> 数据来源：定量指标由 `a-stock-data` Skill 提供（Skill 内部按优先级在 13 个数据源中自动选取，见 §11）；
> 定性分析（标记 `AI`）由大模型生成，并附引用来源与生成时间戳。

---

## 0. 目标与范围

- **一句话目标**：输入一个股票代码，输出一份"是否值得买、当前贵不贵、买点在哪、看多看空理由"的结构化研究报告。
- **核心问题**：买什么（质量）→ 值多少（内在价值估值）→ 何时买（合理价值 − 安全边际 = 击球区）→ 为什么（多空逻辑）。
- **页面路由（建议）**：`/stocks/[code]`，例如 `/stocks/601088`（中国神华）。
- **数据标注约定**：
  - 🟦 `Data`：由 `a-stock-data` Skill 提供的客观数据，可回测、可量化。
  - 🟨 `AI`：模型生成的定性判断，需标注「生成时间 + 信息来源 + 置信度」，不可作为唯一决策依据。

## 2. 业务分析（Business）🟨 AI + 🟦 Data

**分析目标**：看懂公司靠什么赚钱、收入结构是否健康。

| 维度 | 内容 | 来源 |
|------|------|------|
| 业务模式 | Top3 业务线及其营收占比、毛利占比 | 🟨 AI（财报）+ 🟦 营收数据 |
| 收入分析 | 国内 / 国外收入拆分与同比趋势 | 🟨 AI（财报）|

**页面功能**：
- Top3 业务收入占比环形图 + 近 3~5 年趋势堆叠柱状图。
- 国内/海外收入双环对比，标注同比增速。

**UI 设计**：
- 双列卡片：左「业务结构」环形图，右「地区结构」环形图。
- 鼠标悬停显示金额、占比、同比；AI 段落给出一句话定性结论。

## 9.5 企业综合评分（Company Scorecard）🟨 AI ⭐

**位置**：置于「估值总结」之前——先回答「是一家什么样的公司」，再回答「值多少钱」。对应 `PRD_0.md` 击球区的三方面拆解。

**分析目标**：用统一口径把公司浓缩为三维评分，作为后续估值方法选择与安全边际的定性依据。

**三个维度（0~5 分，口径统一「越高越利于价值投资」）**：

| 维度 | 子项（来自 PRD_0.md） | 说明 |
|------|----------------------|------|
| **稳定性** | 商业模式 + 护城河 / 分红稳定 / 财务状况 | 越高越稳健（综合 §6 股东回报、§8 护城河、§5 财务） |
| **成长性** | 行业空间 / 资本开支 / 技术创新 | 越高成长性越强（综合 §2 业务、§9 全球化） |
| **周期性** | 行业周期 / 股价周期 | **反向口径**：分数越高代表周期性越弱 / 穿越周期确定性越强（综合 §4 周期性）；强周期股得低分 |

**输出**：
- 三维各自的综合分 + 子项评分（带依据）+ 维度小结。
- 综合评分（三维平均）+ 企业画像标签（如「高股息 · 类债周期龙头」「强周期 · 高弹性航运」）+ 一段总评。

**与估值的联动**：
- 高稳定 + 弱周期 + 分红稳 → 倾向股息率法（①）。
- 有成长、分红不高 → 倾向 DCF（②）。
- 强周期 / 转型期、当前盈利失真 → 倾向未来盈利能力法（③），并要求更大安全边际。

**UI 设计**：
- 顶部综合评分大数 + 画像标签 + 三维雷达图（三角形直观看出公司画像）。
- 下方三张维度卡片：维度分 + 评级 chip + 子项迷你进度条 + 依据 + 维度小结。

**生成与对齐**：🟨 AI 定性主导，与股票池对齐（仅池内标的生成），开发期由 Cursor 综合各模块后写入 `data/scorecard/{code}.json`，API `/api/stock-scorecard`。

## 10. 估值总结（Valuation Summary）🟦 Data + 🟨 AI ⭐核心输出

**分析目标**：估算公司「内在价值」，判断现价相对内在价值贵不贵、买点在哪。

**核心原则**：
- **PE / PB / 股息率历史分位只作参考锚点，不作为估值结论**。分位只反映相对自身历史的位置，不等于贵贱——周期股在盈利高点会显得"低 PE"、转型股盈利失真会让 PE 完全失效，单看分位极易误判。
- **估值以"内在价值"为锚**：先按公司特征选择合适的估值方法算出合理价值，再叠加安全边际得到击球区买入价。

### 三种估值方法

| 方法 | 适用对象 | 估值逻辑 | 来源 |
|------|---------|---------|------|
| **① 股息率估值** | 成长平滑（不倒退）、分红稳定持续的公司 | 用可持续股息 ÷ 目标股息率，倒推合理价。**忽略成长**：把当前可持续分红当作「无增长永续」资本化，目标股息率只反映风险（自身历史中枢 + 无风险利率 + 风险溢价），不因成长压低。成长性留到「估值总结 / 推荐 / 看多逻辑」体现 | 🟦 分红序列 + 🟨 目标股息率（风险视角） |
| **② DCF 自由现金流** | 有一定成长、但分红不高的公司 | 预测未来自由现金流并折现求企业价值，扣净债得股权价值 / 每股内在价值 | 🟦 历史 FCF + 🟨 增长/折现/永续假设 |
| **③ 未来盈利能力估值** | 商业模式处于转型期、当前盈利不能代表常态的公司 | 估计转型完成后的稳态盈利（或盈利能力），乘以合理倍数得目标市值 | 🟨 AI 主导 + 🟦 佐证 |

**应用规则**：
- **每家公司至少给出方法 ① 和方法 ② 两个估值结果**，并列展示便于交叉验证。
- **若公司符合方法 ③ 特征**（转型期 / 当前盈利失真），**额外给出方法 ③** 的估值。
- 各方法输出「合理价值」（**固定单点值**）+ 关键假设；不适用的方法置灰并注明原因。
- **方法 ① 的目标股息率须显式给出公式**：`目标股息率 = 无风险利率 + 风险溢价`，并标明风险溢价的定档依据（分红稳定性 / 周期性 / 确定性）；风险越高 → 目标股息率越高 → 合理价越低。

**AI 推荐**：
- AI 综合公司特征（成长性、分红稳定性、周期性、是否转型）**推荐主用哪一种方法**，并给出理由与置信度；其余方法作为交叉验证 / 参考。
- **成长性在此环节体现**：股息率法（①）刻意忽略成长、偏保守；若公司有成长，其与 DCF（②）/盈利能力（③）的差额即「成长溢价」，AI 应在推荐理由与看多逻辑中说明，并据此倾向 ②/③ 作为成长股的主估值。

### 击球区（基于"推荐估值 + 安全边际"）

- **击球价 = 推荐方法的合理价值 ×（1 − 安全边际）**。
- **安全边际取值规则**（①②按方法固定，③按确定性分级）：
  - **① 股息率法：固定打 7 折（安全边际 30%）** —— 现金流确定性高。
  - **② DCF 自由现金流法：固定打 6 折（安全边际 40%）** —— 对增长/折现假设敏感。
  - **③ 未来盈利能力法：按确定性分级**（折扣不固定，逐股设定）：
    - 周期股 / 转型股 / 假设敏感度高：40%+（打 6 折或更低）
    - 科技股 / 盈利能力高度不确定：50% ~ 80%（打 5 ~ 2 折）
- **信号灯**（以现价对比）：
  - 🟢 击球区：现价 ≤ 击球价
  - 🟡 合理：击球价 < 现价 ≤ 合理价值
  - 🔴 高估（观望）：现价 > 合理价值

| 输出项 | 说明 | 来源 |
|--------|------|------|
| 三种估值 | ① 股息率 / ② DCF /（③ 转型盈利）各自合理价值 | 🟦 + 🟨 |
| 推荐方法 | AI 选定的主估值方法 + 理由 + 置信度 | 🟨 AI |
| 合理价值 | 推荐方法给出的内在价值（固定单点值） | 🟦 + 🟨 |
| 安全边际 | 按确定性设定的折扣 | 🟨 规则 |
| 击球价 | 合理价值 ×（1 − 安全边际） | 🟦 规则 |
| 估值分位（参考） | PE / 股息率历史分位，仅作参考锚点，不作结论 | 🟦 参考 |
| 看多逻辑 | 3~5 条买入理由 | 🟨 AI（综合各模块）|
| 看空逻辑 | 3~5 条风险/卖出理由 | 🟨 AI（综合各模块）|

### 看多 / 看空逻辑（保留）

- **看多逻辑**：3~5 条买入理由，由 AI 综合 §2~§9 各模块（业务、运营、利润质量、财务、股东回报、政策、护城河、全球化）归纳。
- **看空逻辑**：3~5 条风险 / 卖出理由，同样综合各模块的负面信号与估值假设敏感点。
- 每条结论带**来源锚点**，点击跳转对应分析模块，保证多空判断可溯源、可验证。
- 与估值收敛联动：看多 / 看空逻辑应能解释「为什么推荐这种估值方法、为什么给这个安全边际」。

**页面功能**：
- **三种估值方法卡片并排**：每张含「方法名 / 适用性标签 / 合理价值 / 关键假设」；推荐方法高亮加徽章，不适用方法置灰。
- **击球区面板**：信号灯（绿/黄/红）+ 现价 / 击球价 / 合理价值 三值对比 + 安全边际标注。
- PE-TTM / 股息率历史分位图保留，但显式标注「**仅供参考，非估值依据**」。
- 看多 / 看空逻辑左右双栏卡片，每条带来源锚点。
- 「加入选股池」「设置价格预警」操作按钮。

**UI 设计**：
- 顶部一个醒目的「击球区信号灯」（绿/黄/红）+ 现价 vs 击球价。
- 估值三方法并排卡片，推荐方法加「⭐ AI 推荐」徽章并附一句话理由。
- 看多绿色卡 / 看空红色卡并排，每条带来源锚点（点击跳转对应模块）。
